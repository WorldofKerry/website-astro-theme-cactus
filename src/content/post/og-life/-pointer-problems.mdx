# Paramaterized Primes

Say you want to write a function that returns all the prime numbers within a range of numbers, and you're given a correct `isPrime` function. How would you write this function?

```
int isPrime(int n) {
    /* ... */
}
```

```
int* getPrimesInRange(int start, int end, int* size) {
    int *arr = (int*)malloc((end - start) * sizeof(int));
    int next = 0;
    for (int i = start; i < end; i++) {
        if (isPrime(i)) {
            arr[next++] = i;
        }
    }
    *size = next;
    arr = (int*)realloc(arr, next * sizeof(int));
    return arr;
}
```
